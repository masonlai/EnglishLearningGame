# EnglishLearningGame
## This is a simple project which uses Tkinter as GUI.

## Features

- Three games for learning English:
  1.	Spelling game | guessing the missing character from the word 
  2.	Reorganise word | spelling the word by correct order based on a unorder word
  3.	Recognize word | choosing the correct answers from three words based on the definition  
- Word definition provided
- Ranking
- A cute cat
